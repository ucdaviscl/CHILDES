# CHILDES

Investigating prepositional phrase ordering as part of the child language development process in terms of dependency shortening and unigram distributions. Data comes from the [Eng-NA](http://childes.talkbank.org/access/Eng-NA/), [Japanese](http://childes.talkbank.org/access/Japanese/), and [Spanish](http://childes.talkbank.org/access/Spanish/) corpora of the [CHILDES](http://childes.talkbank.org/) database.

## Built With

* [Perl](https://www.perl.org/) 
* [Python](https://www.python.org/)
